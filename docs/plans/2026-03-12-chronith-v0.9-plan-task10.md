## Step 10.9: Monitor CI and review comments

After the PR is created, poll for CI status:

```bash
gh pr checks <PR_NUMBER> --watch
```

If any check fails, fix the issue, re-run `act` locally to confirm the fix, then push.

Poll for review comments:

```bash
gh api repos/juliusbartolome/chronith/pulls/<PR_NUMBER>/comments
gh pr view <PR_NUMBER> --json reviews
```

Address each comment, push the fix, and reply:

```bash
gh api repos/juliusbartolome/chronith/pulls/<PR_NUMBER>/comments/<COMMENT_ID>/replies \
  -f body="Fixed in <commit-sha>. <brief explanation>"
```

Repeat until all CI checks are green and all review comments are resolved.

---

## Step 10.10: Merge the PR

Once all checks are green and reviews are resolved:

```bash
gh pr merge <PR_NUMBER> --merge --delete-branch
```
