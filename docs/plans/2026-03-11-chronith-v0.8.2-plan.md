# Chronith v0.8.2 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Wire the dead-code OTel spans and metrics into real call sites, add a missing ApiKeyAgingService background service, and fix a stale Swagger version string — all three were omitted from v0.8.

**Architecture:** No new interfaces needed. `ChronithActivitySource` is a static class called directly at call sites; `ChronithMetrics` is a singleton that gets constructor-injected into handlers and background services. `ApiKeyAgingService` follows the `AuditRetentionService` pattern: scoped DI inside a `BackgroundService`, with `IApiKeyRepository` queried per-tenant using `ListByTenantAsync`, and warnings written via `IAuditEntryRepository`.

**Tech Stack:** .NET 10, MediatR 14, xUnit, FluentAssertions, NSubstitute

**Key facts:**

- `ChronithActivitySource` is a **static class** — no DI injection needed, just call the static methods directly.
- All methods return `Activity?` — wrap call sites in `using var activity = ChronithActivitySource.Start...(...)`.
- `Activity` implements `IDisposable`; wrapping in `using` ensures the span is ended correctly.
- The Application layer must reference `Chronith.Infrastructure` for `ChronithActivitySource`. The class is already in `Chronith.Infrastructure.Telemetry`, but the Application commands currently do not reference Infrastructure. **Solution:** move `ChronithActivitySource` is NOT in the Application layer — instead, call it from the Infrastructure-layer services (`WebhookDispatcherService`, `NotificationDispatcherService`, `RecurringBookingGeneratorService`) and for the MediatR handlers (which compile in the Application project), call the activity source only in the Infrastructure **pipeline behavior** or accept the cross-layer call. Re-check csproj references before implementing.

> 1. Move `ChronithActivitySource` to `Chronith.Application` — violates "no framework dependencies" rule (it uses `System.Diagnostics` which is a BCL namespace, not a third-party framework, so this is acceptable).
> 2. Only wire spans in Infrastructure services, not Application handlers.

```csharp
using Chronith.Application.Interfaces;
using Microsoft.Extensions.DependencyInjection;
using Microsoft.Extensions.Hosting;
using Microsoft.Extensions.Logging;
using Microsoft.Extensions.Options;

namespace Chronith.Infrastructure.Services;

public sealed class ApiKeyAgingService(
    IServiceScopeFactory scopeFactory,
    IOptions<ApiKeyAgingOptions> options,
    IBackgroundServiceHealthTracker healthTracker,
    ILogger<ApiKeyAgingService> logger)
    : BackgroundService
{
    protected override async Task ExecuteAsync(CancellationToken stoppingToken)
    {
        while (!stoppingToken.IsCancellationRequested)
        {
            try
            {
                await CheckAgingAsync(stoppingToken);
                healthTracker.RecordSuccess(nameof(ApiKeyAgingService));
            }
            catch (Exception ex) when (ex is not OperationCanceledException)
            {
                logger.LogError(ex, "Error during API key aging check");
            }

            await Task.Delay(TimeSpan.FromHours(options.Value.CheckIntervalHours), stoppingToken);
        }
    }

    // internal for unit testing via InternalsVisibleTo
    internal async Task CheckAgingAsync(CancellationToken ct)
    {
        using var scope = scopeFactory.CreateScope();
        var tenantRepo = scope.ServiceProvider.GetRequiredService<ITenantRepository>();
        var apiKeyRepo = scope.ServiceProvider.GetRequiredService<IApiKeyRepository>();

        var tenants = await tenantRepo.ListAllAsync(ct);
        if (tenants.Count == 0)
        {
            logger.LogDebug("No tenants found; skipping API key aging check");
            return;
        }

        var now = DateTimeOffset.UtcNow;
        var threshold = TimeSpan.FromDays(options.Value.ThresholdDays);

        foreach (var tenant in tenants)
        {
            try
            {
                var keys = await apiKeyRepo.ListByTenantAsync(tenant.Id, ct);
                foreach (var key in keys)
                {
                    if (key.IsRevoked || key.IsExpired(now))
                        continue;

                    var age = now - key.CreatedAt;
                    if (age >= threshold)
                    {
                        logger.LogWarning(
                            "API key {KeyId} (tenant {TenantId}, description: '{Description}') is {AgeDays} days old (threshold: {ThresholdDays} days). Consider rotating it.",
                            key.Id,
                            key.TenantId,
                            key.Description,
                            (int)age.TotalDays,
                            options.Value.ThresholdDays);
                    }
                }
            }
            catch (Exception ex) when (ex is not OperationCanceledException)
            {
                logger.LogError(ex, "Error checking API key aging for tenant {TenantId}", tenant.Id);
            }
        }
    }
}
```
