# Chronith v0.8 — Production Infrastructure — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Harden Chronith for production with audit logging, OpenTelemetry observability, security hardening (CORS, CSP, rate limiting redesign, secrets management), database optimization, and customizable notification templates.

**Architecture:** Five independent pillars — (1) Audit Logging adds `AuditEntry` domain model with `AuditBehavior` MediatR pipeline that captures before/after snapshots of mutated entities via `IAuditSnapshotResolver`; (2) Observability adds OTel tracing/metrics with custom spans and meters, Serilog enrichers, correlation ID middleware, and enhanced health checks; (3) Security Hardening adds CORS, CSP, redesigned rate limiting (4 named policies replacing global limiter), input sanitization, and secrets management (AES-256-GCM encryption, multi-key JWT, API key expiry); (4) Database Optimization adds composite indexes, query tags, split queries, and PgBouncer; (5) Notification Templates adds customizable per-tenant templates with variable substitution replacing hardcoded notification bodies.

**Tech Stack:** .NET 10, FastEndpoints 8.x, MediatR 14.x, FluentValidation 12.x, EF Core 10.x + Npgsql, PostgreSQL 17, Redis 8, OpenTelemetry, Serilog, AES-256-GCM (System.Security.Cryptography), xUnit + FluentAssertions + NSubstitute + Testcontainers.

**Design Doc:** `docs/plans/2026-03-07-chronith-v0.8-design.md`

---

## Key Architectural Decisions

1. **AuditBehavior snapshot approach:** Commands implement `IAuditable` with `EntityId`, `EntityType`, `Action` properties. The behavior resolves entity state before and after handler execution (two DB round-trips — accepted overhead).
2. **Rate limiting redesign:** 4 named policies (`Authenticated`/300/1min/tenant, `Public`/60/1min/IP, `Auth`/10/5min/IP, `Export`/5/1min/tenant) replace the single global limiter. Endpoints opt-in via `Options(x => x.RequireRateLimiting("PolicyName"))`.
3. **Notification credential encryption:** AES-256-GCM at rest. `TenantNotificationConfigEntity.Settings` column type changes from `jsonb` to `text` storing encrypted base64.
4. **OTel local stack:** Jaeger + Prometheus + Grafana added to `docker-compose.yml`.

## Codebase Reference

- **ITenantContext** already has `TenantId`, `UserId`, `Role` — no new auth interface needed for audit.
- **MediatR pipeline order:** `LoggingBehavior` → `ValidationBehavior` → `PerformanceBehavior`. `AuditBehavior` inserts after `ValidationBehavior`, before `PerformanceBehavior`.
- **IQuery marker** lives in `PerformanceBehavior.cs` — not a separate file.
- **Program.cs rate limiting:** Single `GlobalLimiter` with `PartitionedRateLimiter`. Redesign replaces with `AddPolicy()` calls.
- **NotificationDispatcherService** uses hardcoded body. Templates replace this.
- **TenantNotificationConfig.Settings** is stored as JSONB. Must change to encrypted text.
- **TenantApiKeyEntity** has no `ExpiresAt` field — needs adding.
- **JwtTokenService** reads single `Jwt:SigningKey` — needs multi-key support.
- **docker-compose.yml** has postgres + redis + chronith-api only.
- **CORS/CSP:** Not configured — adding from scratch.

### Step 1.11: Implement AuditBehavior

Create `src/Chronith.Application/Behaviors/AuditBehavior.cs`:

```csharp
public sealed class AuditBehavior<TRequest, TResponse>(
    IEnumerable<IAuditSnapshotResolver> resolvers,
    IAuditEntryRepository auditRepository,
    ITenantContext tenantContext,
    IUnitOfWork unitOfWork
) : IPipelineBehavior<TRequest, TResponse>
    where TRequest : notnull
{
    public async Task<TResponse> Handle(TRequest request, RequestHandlerDelegate<TResponse> next, CancellationToken ct)
    {
        if (request is not IAuditable auditable)
            return await next();

        var resolver = resolvers.FirstOrDefault(r => r.EntityType == auditable.EntityType);
        var oldValues = resolver is not null
            ? await resolver.ResolveSnapshotAsync(auditable.EntityId, ct)
            : null;

        var response = await next();

        var newValues = resolver is not null
            ? await resolver.ResolveSnapshotAsync(auditable.EntityId, ct)
            : null;

        var entry = AuditEntry.Create(
            tenantContext.TenantId,
            tenantContext.UserId,
            tenantContext.Role,
            auditable.EntityType,
            auditable.EntityId,
            auditable.Action,
            oldValues,
            newValues,
            null);

        await auditRepository.AddAsync(entry, ct);
        await unitOfWork.SaveChangesAsync(ct);

        return response;
    }
}
```

**Implementation** using `System.Security.Cryptography.AesGcm`:

```csharp
public sealed class EncryptionService : IEncryptionService
{
    private readonly byte[] _key;

    public EncryptionService(IOptions<EncryptionOptions> options)
    {
        _key = Convert.FromBase64String(options.Value.MasterKey);
    }

    public string Encrypt(string plaintext)
    {
        var nonce = new byte[AesGcm.NonceByteSizes.MaxSize]; // 12 bytes
        RandomNumberGenerator.Fill(nonce);
        var plaintextBytes = Encoding.UTF8.GetBytes(plaintext);
        var ciphertext = new byte[plaintextBytes.Length];
        var tag = new byte[AesGcm.TagByteSizes.MaxSize]; // 16 bytes

        using var aes = new AesGcm(_key, AesGcm.TagByteSizes.MaxSize);
        aes.Encrypt(nonce, plaintextBytes, ciphertext, tag);

        // Format: base64(nonce + ciphertext + tag)
        var result = new byte[nonce.Length + ciphertext.Length + tag.Length];
        nonce.CopyTo(result, 0);
        ciphertext.CopyTo(result, nonce.Length);
        tag.CopyTo(result, nonce.Length + ciphertext.Length);
        return Convert.ToBase64String(result);
    }

    public string Decrypt(string encoded)
    {
        var data = Convert.FromBase64String(encoded);
        var nonce = data[..AesGcm.NonceByteSizes.MaxSize];
        var tag = data[^AesGcm.TagByteSizes.MaxSize..];
        var ciphertext = data[AesGcm.NonceByteSizes.MaxSize..^AesGcm.TagByteSizes.MaxSize];
        var plaintext = new byte[ciphertext.Length];

        using var aes = new AesGcm(_key, AesGcm.TagByteSizes.MaxSize);
        aes.Decrypt(nonce, ciphertext, tag, plaintext);
        return Encoding.UTF8.GetString(plaintext);
    }
}
```

### Step 11.4: Implement ITemplateRenderer and TemplateRenderer

```csharp
public interface ITemplateRenderer
{
    string Render(string template, Dictionary<string, string> context);
}

public sealed class TemplateRenderer : ITemplateRenderer
{
    public string Render(string template, Dictionary<string, string> context)
    {
        if (string.IsNullOrEmpty(template)) return template;

        var result = template;
        foreach (var (key, value) in context)
        {
            result = result.Replace($"{{{{{key}}}}}", value ?? string.Empty);
        }
        return result;
    }
}
```
